# Web Template

A simple and reusable web template for quickly building and deploying websites. Designed for easy customization and GitHub Pages deployment.

---

## Features

- Clean and minimal design
- Easy to customize (HTML / CSS / JavaScript)
- No build tools required
- Ready for GitHub Pages
- Friendly
